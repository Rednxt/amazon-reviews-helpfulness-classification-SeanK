DATASET TIDAK DISERTAKAN KARENA BATASAN UKURAN GITHUB (100MB+).

Untuk menjalankan kode dalam repositori ini, dataset akan diunduh secara otomatis menggunakan Kaggle API di dalam notebook Google Colab.

Sumber Dataset: 
https://www.kaggle.com/datasets/mohamedbakhet/amazon-books-reviews

Cara kerja di Colab:
1. Upload file `kaggle.json` (API Token Kaggle Anda) ke sesi Colab.
2. Jalankan sel kode berikut di notebook:
   !pip install -q kaggle
   !mkdir -p ~/.kaggle
   !cp kaggle.json ~/.kaggle/
   !chmod 600 ~/.kaggle/kaggle.json
   !kaggle datasets download -d mohamedbakhet/amazon-books-reviews
   !unzip amazon-books-reviews.zip
